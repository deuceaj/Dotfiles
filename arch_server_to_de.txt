Add display manager
    yay -S sddm

Enable SDDM
    sudo systemctl enable sddm

Change from CLI target to Graphical targer
    sudo systemctl set-default graphical.target

Install my config files
    git clone https://github.com/deuceaj/Dotfiles

Copy Config to .Config
    mv Dotfiles/config/* to .config/

Install required packages
    yay -S awesome fed firefox-developer-edition kitty polybar nitrogen neofetch  ocs-url picom rofi sxhkd thunar visual-studio-code-bin

Autologin
    sudo nano /etc/sddm.comg
